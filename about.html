Add tag with id="gen-img-status" to the header of each page. This is where the main logic is executed and the transformation status is shown. When the picture is transformed the status changes. (On this page it is a block with a red border)

<div id="gen-img-status"></div>

Add tag with id="gen-img-content" to the desired page where you want to show the transformation screen.

<div id="gen-img-content"></div>
